# Used Car Price Prediction

This project predicts the resale value of used cars using Machine Learning.

Dataset:
CarDekho Vehicle Dataset

Features
- Year
- Fuel Type
- Transmission
- Owner
- Mileage

Feature Engineering
CarAge = CurrentYear - Year

Models Used
- Random Forest
- Gradient Boosting
- XGBoost

Evaluation Metrics
- RMSE
- R² Score

Tools
Python
Scikit-learn
XGBoost
Google Colab
